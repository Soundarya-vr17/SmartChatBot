# 🤖 Smart AI Chatbot

A beginner-friendly chatbot developed using **Python** and **CustomTkinter**. This project provides a simple graphical user interface where users can ask predefined questions and receive instant responses.

## ✨ Features

- 💬 Simple and user-friendly chatbot interface
- 🤖 Responds to predefined questions
- 💾 Saves chat history to a text file
- 🌙 Dark theme interface
- 🖥️ Built using Python and CustomTkinter

## 🛠️ Technologies Used

- Python
- CustomTkinter
- File Handling

## 📂 Project Files

- `app.py` – Main chatbot application
- `chat_history.txt` – Stores conversation history
- `README.md` – Project documentation

## ▶️ How to Run the Project

1. Download or clone this repository.
2. Install the required library:

```bash
pip install customtkinter
```

3. Run the application:

```bash
py app.py
```
